Le projet PHP doit etre ouvert via un serveur local.

Si tu vois cette page en ouvrant un fichier directement, c est normal : le navigateur ne peut pas executer PHP en mode file://.

Lance un serveur local avec php -S 127.0.0.1:8000 -t Projet_echec2 puis ouvre http://127.0.0.1:8000/index.php.

Si la page est deja servie en http:// ou https://, tu vas etre redirige automatiquement vers index.php.
